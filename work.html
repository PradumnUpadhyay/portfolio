TorrGrab

Preview GitHub

Created my own torrent client CLI based in Python3. Simply execute the file in terminal and search for what you want to download, based on that query a list of results will be displayed inside terminal from different torrent sites, to download just specify the serial number and you are done. Technologies used are: Python3, urllib, BeautifulSoup bs4 module, requests and os library
